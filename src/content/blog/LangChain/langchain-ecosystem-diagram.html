LangChain 生态定位

根据需求复杂度选择合适的工具层次

Deep Agents（推荐起点）
  • 开箱即用的完整 Agent 框架
  • 自动压缩长对话、虚拟文件系统
  • 支持子 Agent 生成和上下文隔离
↓
LangChain（本框架）
  • 快速构建 Agent 和自主应用
  • 灵活的上下文工程能力
  • 基于 LangGraph 构建
↓
LangGraph（底层编排框架）
  • 低级别 Agent 编排和运行时
  • 支持确定性和 Agent 工作流混合
  • 需要深度定制时使用
选择建议
快速构建 Agent → 使用 LangChain
需要现代功能（自动压缩对话、子 Agent） → 使用 Deep Agents
高级定制需求 → 使用 LangGraph
