# 每日热点日报功能

在博客网站实现每日拉取全球热点并生成日报展示的功能。

## 技术方案

### 技术栈
- **部署平台**: Vercel（Serverless Functions + Cron Jobs）
- **数据源**: NewsAPI.org（免费版每天 100 次请求）
- **数据存储**: 内存缓存

### 架构设计

```
┌─────────────────────────────────────────────────────┐
│                    Vercel Platform                   │
├─────────────────────────────────────────────────────┤
│  ┌─────────────────┐    ┌────────────────────────┐ │
│  │  API Route      │    │  Cron Job               │ │
│  │  /api/news      │    │  每天定时触发           │ │
│  │  (获取热点数据) │    │  → 生成日报 → 缓存     │ │
│  └────────┬────────┘    └───────────┬────────────┘ │
│           │                           │              │
│           ▼                           ▼              │
│  ┌─────────────────────────────────────────────┐   │
│  │  内存缓存                                    │   │
│  └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
                          ▲
                          │ 前端调用
                          ▼
              ┌─────────────────────────┐
              │  前端页面 /daily        │
              │  展示当日日报           │
              └─────────────────────────┘
```

## API 路由

| 路由 | 方法 | 说明 |
|------|------|------|
| `/api/news` | GET | 获取热点新闻（支持缓存） |
| `/api/daily-report` | GET | 生成/获取当日日报 |
| `/api/daily-reports/[date]` | GET | 获取指定日期日报 |

## 前端页面

- **路径**: `/daily`
- **功能**: 展示当日热点日报，支持分类切换、刷新

## 环境配置

在 Vercel 项目设置中添加环境变量：

| 变量名 | 说明 | 获取方式 |
|--------|------|----------|
| `NEWS_API_KEY` | NewsAPI 密钥 | [newsapi.org](https://newsapi.org/register) |

**注意**: 如果未配置 `NEWS_API_KEY`，API 会返回模拟数据。

## Vercel Cron 配置

`vercel.json` 配置了定时任务：
- 每天 UTC 0:00（北京时间 8:00）执行日报生成
- 调用 `/api/daily-report` 接口

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

访问 http://localhost:5173/daily 查看日报页面。

## 部署

1. 将代码推送到 GitHub
2. 在 Vercel 导入项目
3. 添加环境变量 `NEWS_API_KEY`
4. 部署完成
